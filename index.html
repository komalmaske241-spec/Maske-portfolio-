🌐 My Learning Journey — Web Development

Day 1 — HTML Basics

On the 1st day I learned the basics of HTML. I learned how to structure a webpage, explored various HTML tags like headings, paragraphs, bold, italic, underline, lists, marquee, font, and images. I also practiced adding audio and links. My first assignment was creating an SSGMCE info page.

Day 2 — Tables & Links

Day 2 covered HTML tables using <table>, <tr>, <th>, <td> with attributes like border, cellpadding, cellspacing, bgcolor, rowspan, and colspan. I built a marks table and a student info table.

Day 3 — Introduction to CSS Selectors

Day 3 introduced CSS: element selectors (styling all p tags with color: blue; font-size: 18px) and class selectors (using .highlight with brown background and cadetblue text). I practiced writing both in separate HTML files.

Day 4 — CSS Flexbox

Day 4 explored CSS Flexbox layout. I used display: flex, justify-content: space-between, align-items: center, and flex-direction: row to arrange 8 numbered boxes inside a blue-violet container. Each box had dark sea green background and brown text.

Day 5 — Advanced CSS & To-Do App Styling

Day 5 covered advanced CSS: border-radius, box-shadow, padding, margin, and background-color for creating card-like UI. I styled the To-Do List app container with rgb(156,222,252) background, border-radius 15px, and box-shadow. Also added a styled header with font-size 1.7em.

Day 6 — JavaScript Basics

Day 6 introduced JavaScript. I declared variables using var, performed arithmetic operations (subtraction), used comparison operators (>, >>), logical operators, and console.log() to print output. I ran the JS file using Node.js in the VS Code terminal and saw outputs like 100, true, 2, 0, 1.
