🌐 My Learning Journey — Web Development

Day 1 — HTML Basics

On the 1st day I learned the basics of HTML. I learned how to structure a webpage, explored various HTML tags like headings, paragraphs, bold, italic, underline, lists, marquee, font, and images. I also practiced adding audio and external links. My first assignment was creating an SSGMCE information page in VS Code.

HTML Structure <h1> <p> <b> <i> Lists & Links Images & Audio Marquee & Font
📄 View Assignment

Day 2 — Images, Tables & Links

Day 2 covered HTML tables using <table>, <tr>, <th>, <td> with attributes like border, cellpadding, cellspacing, bgcolor, rowspan, and colspan. I built a Branch/Seats/Cut-off table and a Student Name/Age/Gender info table.

<table> <tr> <th> <td> rowspan & colspan cellpadding & cellspacing bgcolor attribute
📄 View Assignment

Day 3 — Introduction to CSS Selectors

Day 3 introduced CSS. I learned element selectors — styling all <p> tags with color: blue and font-size: 18px. I also learned class selectors using .highlight with brown background and cadetblue text color. I practiced both types in separate HTML files.

Element Selector Class Selector color & font-size background-color <style> tag
📄 View Assignment

Day 4 — CSS Flexbox Layout

Day 4 explored CSS Flexbox. I used display: flex, justify-content: space-between, align-items: center, and flex-direction: row to arrange 8 numbered boxes inside a blue-violet container. Each box had dark sea green background and brown text.

display: flex justify-content align-items flex-direction height & width
📄 View Assignment

Day 5 — Advanced CSS & To-Do App Styling

Day 5 covered advanced CSS properties. I styled a To-Do List app container with background-color: rgb(156,222,252), border-radius: 15px, and box-shadow: 4px 5px 6px. The body had display flex with center alignment and grey background. I also added a styled header with font-size 1.7em and font-weight 900.

border-radius box-shadow padding & margin font-weight CSS Box Model
📄 View Assignment (To-Do App)

Day 6 — JavaScript Basics

Day 6 introduced JavaScript. I declared variables using var, performed arithmetic (subtraction: var sub = b - a), used comparison operators (>, >>), logical operators, and console.log() to print output. Running in Node.js terminal gave outputs: 100, true, 2, 0, 1.

var variables console.log() Arithmetic Operators Comparison Operators Node.js Terminal
